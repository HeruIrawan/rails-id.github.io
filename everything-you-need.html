Rails punya apa pun yang kamu butuhkan

Ruby on Rails bukan framework yang minimalist, melainkan framework yang metropolis. Ruby on Rails merupakan sebuah framework yang dipenuhi dengan semua hal yang dibutuhkan oleh institusi besar untuk menjalankan aplikasi yang besar dan berat seperti Basecamp, Github atau Shopify.

Hal ini berarti bahwa banyak hal yang harus dipelajari! Tapi hal ini juga berarti bahwa Kamu dapat mempelajarinya sedikit demi sedikit. Kamu tidak perlu menjadi seorang ahli dalam semua aspek di Ruby on Rails untuk memulai membuat aplikasi yang hebat.

Tulang punggung dari sebuah aplikasi Rails adalah desain Model-View-Controller yang mengatur logika pemrograman menjadi 3 lapisan utama. Model adalah tempat dimana semua logika bisnismu diletakan. Disinilah kamu menghitung apakah invoice selanjutnya sudah kadaluarsa atau berapa banyak tugas yang tersisa untuk diselesaikan dalam suatu project. Controller bertugas untuk menerima click dan tap dari user, mengarahkan perintah tersebut ke model, dan kemudian me-render template HTML atau JSON sebagai View.

Di dalam Rails, model secara umum diatur menggunakan Active Record (dengan beberapa backup menggunakan Active Model). Active Record adalah framework yang digunakan untuk berbicara dengan database. Controller diatur menggunakan Action Controller dan View diatur menggunakan Action View, yang secara bersamaan disebut sebagai Action Pack.

Selain framework inti di atas, kami juga menyediakan Action Mailer, yang digunakan untuk mengirim email dalam bentuk HTML maupun plaintext. Action Cable yag digunakan untuk melakukan live update pada halaman melalui WebSocket. Active Support yang mempercantik bahasa dan standard library Ruby dengan gula dan manisan untuk digunakan dalam aplikasi web. Dan terakhir, Railties yang menyediakan semua command, generator, dan lem untuk merekatkan semua framework ini menjadi suatu sistem terpadu yaitu Ruby on Rails.
